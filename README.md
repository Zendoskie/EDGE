## EDGE - Academic Risk Analysis System

This project is a deep learning-based student academic risk analysis and early warning system built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

### Getting started

```sh
# Install dependencies
npm install

# Start the development server
npm run dev
```

### Deployment

You can deploy this project to platforms such as Vercel, Netlify, or any static hosting provider that supports Vite builds.
